## 😎 개요

- Open Trivia API를 이용한 무작위 상식 퀴즈입니다.
- Start 기능, ReStart 기능
- 문제 선택과 동시에 정답 오답 표시
- 정답 선택 시, Next Question 활성화
- 문제 결과(정답/오답 , 차트 , 소요시간)
- 반응형 디자인

## 😊 기술스택

- React.js
- Open Trivia API
- Styled Component
- Media Query
